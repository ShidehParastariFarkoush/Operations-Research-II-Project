# Operations Research II Projects

This repository contains the three phases of my **Operations Research II** course project, which focuses on the **Berth Allocation Problem (BAP)** and its solution using mathematical optimization techniques.

## Project Overview

The objective of this project is to determine the optimal assignment and scheduling of ships to berths while minimizing:

* Ship unloading costs
* Waiting costs
* Extra berth capacity costs

The problem is formulated as a **Mixed-Integer Linear Programming (MILP)** model with practical constraints, including berth capacities, berth lengths, ship priorities, arrival times, and operational delays.

## Project Phases

### Phase 1 – Mathematical Modeling

* Formulation of the Berth Allocation Problem
* Definition of parameters, decision variables, objective function, and constraints
* Linearization of nonlinear components

### Phase 2 – Python Implementation

* Implementation using **Pyomo** and **Gurobi**
* Data preparation and result visualization
* Sensitivity analysis on key model parameters

### Phase 3 – GAMS Implementation

* Implementation of the optimization model in **GAMS**
* Model solution and result analysis
* Additional sensitivity analyses and generation of output tables and charts
